# Global Converter: валюты + единицы в реальном времени

Красивое и удобное веб-приложение для:
- конвертации валют мира (из доступного списка API),
- конвертации единиц измерения,
- отображения даты и времени в реальном времени,
- автоматической работы в офлайн-режиме на последнем сохранённом курсе.

## Возможности

- **Курсы валют в реальном времени** через `open.er-api.com`.
- **Офлайн-резерв**: если интернет пропал, используются последние сохранённые курсы из `localStorage`.
- **Валюты мира**: список валют формируется из ответа API и доступен для конвертации между любыми двумя валютами.
- **Живые дата и время**: часы обновляются каждую секунду.
- **Конвертер единиц**:
  - длина,
  - вес,
  - температура,
  - площадь.
- **Современный UI**: адаптивный интерфейс, карточки, акценты, удобные селекты и поля ввода.

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript (без фреймворков)
- Web API: `fetch`, `localStorage`, `Intl`

## Запуск

1. Склонируйте репозиторий.
2. Откройте `index.html` в браузере.

> Для реальных курсов нужен интернет. Без интернета приложение автоматически переключится на последний сохранённый кэш.

## Структура

- `index.html` — разметка приложения.
- `styles.css` — стили и адаптивность.
- `script.js` — логика загрузки курсов, кэш, конвертация валют и единиц.
